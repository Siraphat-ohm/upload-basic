# Upload API

## Upload a file to the server

### Request
`POST /upload`
### Response
`200 OK` on success

## How to run ?

### Install dependencies
install dependencies using npm or yarn
***USING NPM***
```bash
npm install
```
***USING YARN***
```bash
yarn install
```

### Run the server
***USING NPM***
```bash
npm start
```
***USING YARN***
```bash
yarn start
```
